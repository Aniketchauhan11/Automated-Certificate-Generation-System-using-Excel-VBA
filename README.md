# Automated-Certificate-Generation-System-using-Excel-VBA
Automated certificate generation solution built with Excel VBA and Microsoft Word. Generates personalized PDF certificates in bulk by dynamically replacing placeholders and exporting individual certificates for each candidate.

## Overview

The AI-Powered Certificate Generation System is an automation project developed using Microsoft Excel VBA and Microsoft Word Automation. This solution eliminates the manual effort involved in creating certificates for multiple candidates by automatically generating personalized PDF certificates in bulk.

The application reads candidate information from an Excel spreadsheet, dynamically updates placeholders within a Word certificate template, and exports each certificate as an individual PDF file.

## Key Features

* Bulk certificate generation
* Dynamic placeholder replacement
* Microsoft Word automation
* PDF export functionality
* File selection dialog for template selection
* Folder selection dialog for output location
* Automated file naming using Candidate ID
* Supports large-scale certificate generation
* Reduces manual work and human errors

## Technologies Used

* Microsoft Excel VBA
* Microsoft Word Object Library
* PDF Export Automation
* File Dialog Automation

## Workflow

1. User selects the certificate template file.

2. User selects the destination folder.

3. Excel VBA reads candidate data from the worksheet.

4. Placeholders such as:

   * <<CANDIDATE NAME>>
   * <<DATE>>
   * <<CERTIFICATE ID>>

   are replaced dynamically.

5. A personalized certificate is generated for each candidate.

6. Each certificate is exported as an individual PDF.

7. PDFs are saved automatically using the Candidate ID naming convention.
